# TODO CLI (A todo that runs on your Terminal)
A simple and fast command-line To-Do application built to manage your daily tasks efficiently — right from your terminal.
Features
  Add Task
  Delete Task
  Display Task
  Saved on mongodb database
# Commands
   todo list (For the whole todo list)
   todo add "Add your todos" (For adding the todo on your database)
   todo delete "Simply copy and paste the todo id" (the todo ids are provided after every todo, when todo list is executed)

#TECHSTACK
Language: Node.js 
Storage: JSON
CLI Framework: Commander.js

# Clone the repository
git clone https://github.com/KhushOS-hub/cli-todo.git

# Navigate into the folder
cd todo-cli

# Install dependencies
npm install   

NOTE: TO STORE THE DATA SIMPLY ADD YOUR "MONGO_URI" to the .env file
